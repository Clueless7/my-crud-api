My very first CRUD API

This project aims to provide a list of my hobbies. This site uses Express js as the backend for the API.

My very first CRUD API using Express js. This is created for testing purposes only. This is created by a noob, so don't be angry.

How to use?

chevron_rightGet all hobbies

Request:
GET https://my-crud-app-1.herokuapp.com/api/hobbies
Output:
[
{
    "id": "1",
    "hobby": "coding"
},
{
    "id": "2",
    "hobby": "sleeping"
},
{
    "id": "3",
    "hobby": "being sad"
}
]

chevron_rightGet one hobby

Requirements:
  • id (number)
    • Request:
    GET https://my-crud-app-1.herokuapp.com/api/hobbies/:id
    Output:
    {
    "id": "1",
    "hobby": "coding"
}

    chevron_rightAdd a hobby

    Requirements:
  • Content-type: application/json on the header
  • Id and hobby on the JSON
    • Request:
    POST https://my-crud-app-1.herokuapp.com/api/hobbies/
Content-type: application/json

{
    "id":"x", // x must be a number
    "hobby":"y" // y must be a string
}

    This will send you a status of 204

    chevron_rightDelete a hobby

    Requirements:
  • id (number)
    • Request:
    DELETE https://my-crud-app-1.herokuapp.com/api/hobbies/:id

    This will send you a status of 204

    chevron_rightUpdate a hobby

    Requirements:
  • Content-type: application/json on the header
  • Hobby key and object on the JSON
    • Request:
    POST http://localhost:*/api/hobbies/
Content-type: application/json

{
    "hobby":"y" // y must be a string
}

    This will send you a status of 204
